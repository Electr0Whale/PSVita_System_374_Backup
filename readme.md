### 原装记忆棒救砖方法

补充，进入安全模式方式：  
彻底关机后，R键 PS键 电源键 一起按住10多秒后进入安全模式。如果不灵，可以尝试先按电源键，再同时按住R和PS键，电源键要稍早那么一点点先按，随即进入安全模式。

电脑上[PSV374（解压密码shipengliang）.rar](https://shipengliang.com/go/bmfdjkp7qur3 "2024-07-05 更新")  
下载好，将里面的PSP2UPDAT.PUP拷贝到data/PSP2/UPDATE/ （如果data下没有PSP2/UPDATE文件夹，可以自行手动添加，要确保固件的存放路径为data/PSP2/UPDATE/ ）  
如果降级前没有放，就需要使用读卡器在电脑上放置系统固件文件。

1.  PSV进入安全模式，选升级系统软件，选择使用PS Vita卡升级  
    ![PSV救砖-1](https://pic.shipengliang.com/wp-content/uploads/2020/04/PSV%E6%95%91%E7%A0%96-1.jpg)
2.  系统会自动识别到升级文件，点继续  
    ![PSV救砖-2](https://pic.shipengliang.com/wp-content/uploads/2020/04/PSV%E6%95%91%E7%A0%96-2.jpg)
3.  PSV即可升级变回3.74系统。  
    ![PSV救砖-3](https://pic.shipengliang.com/wp-content/uploads/2020/04/PSV%E6%95%91%E7%A0%96-3.jpg)

1.  下载并安装QCMA：[Qcma\_setup-0.4.1.exe](https://shipengliang.com/go/8grco3aau26n)
2.  将[PSV374（解压密码shipengliang）.rar](https://shipengliang.com/go/bmfdjkp7qur3 "2024-07-05 更新")  
    下载好，将解压后里面的PSP2UPDAT.PUP拷贝进QCMA中的PSV update文件夹  
    ![PSV救砖-4](https://pic.shipengliang.com/wp-content/uploads/2020/04/PSV%E6%95%91%E7%A0%96-4.jpg)
3.  修改QCMA other选项中的配置如图（因为这里用的是3.74固件，所以version为03.74.000）

4.  PSV进入安全模式，选升级系统软件  
    ![PSV救砖-6](https://pic.shipengliang.com/wp-content/uploads/2020/04/PSV%E6%95%91%E7%A0%96-6.jpg)
5.  连接至电脑进行升级
